Vehicle Rental App

Cross-platform Mobile · React Native · Express.js · MySQL

Mobile Application

End-to-end vehicle rental experience: browse inventory, compare options, pick dates, and confirm bookings. Backed by a custom Express.js API with MySQL handling real-time availability. A deliberate architecture decision — Zustand over Redux — eliminated boilerplate and improved navigation responsiveness. React Query manages server state with automatic background refetch, so availability is always current without manual refresh.

↑ Architecture · Zustand cut state boilerplate by ~40% vs Redux

Kensole — Shoes Wholesale E-commerce Platform

Mobile Application · React Native · Redux · Node.js · MongoDB

Mobile Application

A B2B wholesale e-commerce platform purpose-built for bulk shoe buyers — not adapted from a retail template. React Native + Redux delivers a native mobile experience across Android and iOS; a Node.js/Express API backed by MongoDB handles the catalog and order pipeline with flexibility for frequent inventory changes.

↑ Business Impact · Digitized wholesale ordering workflow for a real client
